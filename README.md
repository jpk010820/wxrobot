# 千寻微信框架-本地服务端(Node.js)
Demo提供了一些常用方法。包括了自动通过好友申请、自动收款、自发回复、监听二维码收款等功能<br>
在使用前，务必替换DLL文件，并确认微信版本为3.6.0.18
